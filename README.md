# ⚡ LVSB Energy Report App

ระบบบันทึกรูปภาพมิเตอร์และ Export รายงาน Excel อัตโนมัติ
สำหรับงานตรวจสอบค่าพลังงาน ท่าอากาศยานสุวรรณภูมิ

---

## วิธีติดตั้งและใช้งาน

### ขั้นตอนที่ 1 — ติดตั้ง Python (ครั้งแรกครั้งเดียว)
ดาวน์โหลด Python จาก https://python.org (เลือก Python 3.10 ขึ้นไป)
**ตอนติดตั้งให้ติ๊ก "Add Python to PATH" ด้วย**

### ขั้นตอนที่ 2 — ติดตั้ง Library (ครั้งแรกครั้งเดียว)
เปิด Command Prompt (cmd) แล้วพิมพ์:

```
pip install flask openpyxl pillow
```

### ขั้นตอนที่ 3 — รัน App
ดับเบิลคลิกไฟล์ **`start.bat`** (Windows)
หรือพิมพ์ใน Terminal:
```
python app.py
```

### ขั้นตอนที่ 4 — เปิดเบราว์เซอร์
ไปที่: **http://localhost:5050**

---

## วิธีใช้งาน

1. **เลือกเดือน/ปี** ที่ต้องการทำรายงาน
2. **คลิกชื่อตู้** (SS5-1-LVSB1 ถึง SS5-3-LVSB1)
3. **กรอกวันที่ตรวจ** และ **ค่า kWh** เดือนก่อน/เดือนนี้
4. **คลิกช่องรูป** เพื่ออัปโหลดรูปถ่ายมิเตอร์ (สูงสุด 12 รูปต่อตู้)
5. **กด "บันทึกตู้นี้"** แล้วทำซ้ำกับตู้ถัดไป
6. **กด "ดาวน์โหลด Excel"** เพื่อรับไฟล์รายงาน

```

```

## หมายเหตุ
- ไม่ต้องเชื่อมต่ออินเทอร์เน็ต

- ไฟล์ Excel ที่ได้จะมีรูปฝังในแต่ละ Sheet จริงๆ
